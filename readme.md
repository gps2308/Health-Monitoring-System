🩺 Smart Health Monitoring Dashboard
A Python-based desktop GUI application integrated with AWS Cloud to monitor patient health parameters in real-time.

🚀 Features
Fetches data (Heart Rate ❤️, SpO₂ 🫁, Temperature 🌡️) from AWS DynamoDB via API Gateway + Lambda

Simple, interactive Tkinter GUI

Cloud-based data storage and retrieval

🛠️ Tech Stack
Python (Tkinter, Requests, Pillow)
AWS Lambda, API Gateway, DynamoDB

🔧 How to Run
Clone this repo:
git clone https://github.com/your-username/health-monitoring-dashboard.git
cd health-monitoring-dashboard

Install dependencies:
pip install -r requirements.txt

Run the app:
python health.py
